# User Flows

## Primary Flow: Explore Projects and Contact
**Goal**: Identify Yeabsira's skills and initiate contact for hiring or freelance.

### Entry Points
- LinkedIn Profile
- GitHub Repository
- Awwwards/Portfolio showcases
- Direct URL

### Steps
1. **Landing on Home (Hero)**
   - User sees: "Experimental Sleek" aesthetic, bold H1, and interactive Bento Grid.
   - User action: Scroll to explore or hover over cells.
   - Next: Project Showcase.

2. **Project Exploration**
   - User sees: Animated grid cells for "Digital Addis", "USP Holding", etc.
   - User action: Hover for video loops/WebGL effects; click for details.
   - Next: Project details (modal or section).

3. **Technical Validation**
   - User sees: Tech Stack cell with infinite marquee (React, Next.js, TS).
   - User action: Observe technical metadata.
   - Next: Contact.

4. **Initiate Contact**
   - User sees: "Available for Freelance" status and Contact CTA.
   - User action: Click CTA or Social links.
   - Next: Email client or LinkedIn message.

### Success State
User sends a message or schedules a meeting.

### Alternative Paths
- **Mobile Path**: User scrolls through a single-column stack of Bento cells.
- **Direct Contact**: User ignores projects and goes straight to the floating dock navigation.

## Secondary Flow: Design & Performance Deep-dive
**Goal**: Evaluate the "Experimental Sleek" implementation and 60fps performance (Marcus's goal).

### Entry Points
- Direct URL (Performance audit focus)

### Steps
1. **Scroll Interaction**
   - User sees: Smooth Lenis scroll and GSAP reveals.
   - User action: Rapid scrolling and stopping to test momentum.
   - Next: Interaction testing.

2. **Micro-interaction Testing**
   - User sees: Pixel art avatar glitching, cell hover effects.
   - User action: Hovering over various elements to check snappiness.
   - Next: Mobile responsiveness check.

3. **Responsiveness Check**
   - User sees: Fluid layout adaptation.
   - User action: Resizing browser or using dev tools.

### Success State
User is impressed by the technical execution and design fidelity, leading to a "High-end talent" classification.
