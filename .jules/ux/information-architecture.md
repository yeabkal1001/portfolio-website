# Information Architecture

## Navigation Structure

### Main Navigation (Floating Dock)
- **Home**: Return to top/Hero.
- **Projects**: Jump to Project Bento Grid.
- **About**: Jump to About/Avatar cell.
- **Contact**: Trigger Contact modal or scroll to footer.

### Footer Navigation
- **Socials**: GitHub, LinkedIn, Behance.
- **Info**: Current Time in Addis Ababa, Location.

### Mobile Navigation
- **Hamburger/Dock**: Simplified floating dock at the bottom of the screen for thumb-friendly navigation.

## Page Hierarchy

**Home (Single Page Bento Layout)**
- **Level 1: Above the Fold (The Impact)**
    - **Hero Cell**: Primary branding (Wide Sans H1), "Experimental Sleek" tagline.
    - **Availability Cell**: Real-time status ("Available for Freelance").
- **Level 2: The Core (The Proof)**
    - **Projects Grid**: 4-6 Bento cells of varying sizes.
        - Digital Addis (Large)
        - USP Holding (Medium)
        - Skin Club (Medium)
        - Meta Portfolio (Small)
    - **Tech Stack Cell**: Interactive infinite marquee of logos and metadata.
- **Level 3: The Personal (The Connection)**
    - **About/Avatar Cell**: Pixel art avatar with hover glitch + bio.
    - **Location Cell**: Dynamic Addis Ababa clock and weather/context.
- **Level 4: The Closing (The Action)**
    - **Contact Cell**: Direct form or quick links.

## Content Organization

- **Above the fold**: Hero branding and immediate value proposition.
- **Below the fold**: Project showcase, technical skills, and personal narrative.
- **Sidebar**: None (Bento Grid uses full-width container).
- **Global Elements**: Floating dock (navigation) and background 3D primitives.
