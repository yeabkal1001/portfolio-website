# Typography System

## Font Families
- **Headings**: "Syne", sans-serif (Wide Sans style)
- **Body**: "Inter", system-ui, sans-serif
- **Mono**: "JetBrains Mono", monospace (Technical metadata)

## Type Scale

### Display (Hero)
- Size: 96px / 6rem
- Weight: 800 (Extra Bold)
- Line height: 1.0
- Letter spacing: -0.04em
- Style: Wide Sans

- Size: 64px / 4rem
- Weight: 700
- Line height: 1.1
- Letter spacing: -0.02em
- Style: Wide Sans

## Responsive Typography
- **Desktop**: Full scale (as defined above).
- **Tablet**: Reduce Display/H1-H3 by ~15%.
- **Mobile**: Reduce Display/H1-H3 by ~25%. Body sizes remain consistent for legibility.
