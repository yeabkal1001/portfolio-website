# Color Palette

## Primary Colors
- **Primary**: `#050505` - "Deep Void" (Base Page Background)
- **Primary Light**: `#1A1A1A` - "Elevated Surface" (Bento Cell Background)
- **Primary Dark**: `#000000` - "Pure Black" (Deepest Shadow)

## Secondary Colors
- **Secondary**: `#111111` - "Soft Dark"
- **Secondary Light**: `#222222`
- **Secondary Dark**: `#0A0A0A`

## Accent Colors
- **Accent**: `#ccff00` - "Addis Neon" (Primary Highlight, CTAs)
- **Accent Light**: `#d9ff33` (Hover states)
- **Accent Dark**: `#aacc00` (Active states)
- **Success**: `#00FF94` - "Binary Green"
- **Warning**: `#FFB800` - "Caution Amber"
- **Error**: `#FF3E3E` - "Alert Red"

## Neutral Colors
- **Black**: `#000000`
- **White**: `#FFFFFF`
- **Gray 900**: `#F5F5F5` - Primary text (Off-white for readability)
- **Gray 700**: `#B0B0B0` - Secondary text
- **Gray 500**: `#707070` - Tertiary text
- **Gray 300**: `#333333` - Borders/Dividers
- **Gray 100**: `#1A1A1A` - Component backgrounds
- **Gray 50**: `#050505` - Page background

## Usage Guidelines
- **Primary buttons**: Addis Neon (`#ccff00`) with Black text (`#000000`)
- **Links**: Addis Neon (`#ccff00`)
- **Backgrounds**: Deep Void (`#050505`) for page, Elevated Surface (`#1A1A1A`) for cards
- **Text**: Gray 900 (`#F5F5F5`) for maximum contrast in dark mode
- **Borders**: Gray 300 (`#333333`)
