# Spacing System (8px base unit)

- **xs**: 4px (0.25rem)
- **sm**: 8px (0.5rem)
- **md**: 16px (1rem)
- **lg**: 24px (1.5rem)
- **xl**: 32px (2rem)
- **2xl**: 48px (3rem)
- **3xl**: 64px (4rem)
- **4xl**: 96px (6rem)

## Section Spacing
- Small sections: 64px vertical padding
- Medium sections: 96px vertical padding
- Large sections: 160px vertical padding

## Component Spacing
- Card padding: 32px (Large), 24px (Standard)
- Button padding: 12px 24px
- Input padding: 12px 16px
- Gap between Bento cells: 24px (Desktop), 16px (Mobile)
- Internal component gaps: 16px-24px
