# Quality Standards

These standards must be met for all work on the Yeabsira Kayel Personal Portfolio.

## Performance & Accessibility
- **Lighthouse Score**: 90+ in Performance, Accessibility, Best Practices, and SEO.
- **Mobile-First**: Design and functionality must be optimized for mobile devices first.
- **WCAG 2.1 AA**: Compliance with WCAG 2.1 AA accessibility standards.

## Technical Excellence
- **TypeScript**: Use TypeScript for all code to ensure type safety.
- **Animation**: Achieve 60fps animations for a smooth user experience.
- **Semantic HTML**: Use semantic HTML5 elements for structure and accessibility.
- **Component-Based**: Maintain clean, reusable component-based architecture.

## Visual & Motion
- **Awwwards Quality**: Visuals and interactions should meet Awwwards-winning standards.
- **Smooth Scroll**: Implement Lenis Scroll for momentum-based scrolling.
